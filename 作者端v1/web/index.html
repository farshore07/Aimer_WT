作者端说明与使用指南

这是作者端的第一个版本，为了方便作者们适配语音包而编写，但可能存在BUG，遇到BUG和有建议都可以直接向我反馈，未来UI和布局也都可能会有进一步的更改。

1. 语音包管理

在“语音包库”中可创建、重命名、删除语音包目录，并快速打开语音包库路径。

2. 信息编辑

在“语音包信息”中填写名称、作者、版本、标签、链接、作者留言与版本说明等内容。

3. 试听与关联

主语音包与关联语音包都支持试听音频与封面图，发布前请确认链接可访问、素材可播放。

4. 导入与导出

支持导入 .bank/.zip，也可将当前语音包导出为标准化命名的 .bank 文件。

5. 数据存储

语音包元数据写入 info.json，预览媒体文件写入语音包目录中的 _preview_assets。
